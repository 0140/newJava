
1) ArrayList is not synchronized.	Vector is synchronized.

2) ArrayList is fast because it is non-synchronized.	
Vector is slow because it is synchronized,

3) ArrayList is not a legacy class. It is introduced in JDK 1.2.	
Vector is a legacy class.

4) ArrayList uses the Iterator interface to traverse the elements.
	A Vector can use the Iterator interface 
	or Enumeration interface to traverse the elements.
	
5) ArrayList is Fail fast.
     Vector is fail safe.
