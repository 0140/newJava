this keyword in java 
In java, this is a reference variable that refers to the current object.

Usage of java this keyword

 
this can be used to refer current object  instance variable. ex: this.name = "";
this can be used to invoke current class method (implicitly) ex: this.setOwner("name");
this() can be used to invoke current class constructor.
This() must be the first line in the constructor if we are going to call this().
 
this can be used to return the current class instance from the method.
     ex: return this;
